# Channels

## Telegram rollout

1. add token to env
2. merge channel overlay
3. start gateway
4. approve first DM pairing
5. switch to durable allowlist for owner accounts

## WhatsApp rollout

1. use a dedicated number
2. start in pairing mode
3. approve first owner DM
4. switch to durable allowlist
5. keep groups disabled until needed

## Allowlist automation

Use `clawops allowlists` to render durable channel fragments from a YAML or JSON source list.
